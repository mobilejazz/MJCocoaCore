# MJCocoaCore

## How to 

To install the MJCocoaCore library, just paste the following line in your podfile:
```
pod 'MJCocoaCore', :git => 'https://github.com/mobilejazz/MJCocoaCore.git', :tag => '0.1.5'
```

## Dependencies

MJCocoaCore does not have any dependencies.
